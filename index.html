  • createCanvas(w, h) : Create a new canvas

[parenthesis] width: w height: h

Draw something

  • point(x, y) : Draw a point

[parenthesis] position: (x,y)

  • line(x, y, x2, y2) : Draw a line

[parenthesis] position: (x,y) to (x2,y2)

  • rect(x, y, w, h) : Draw a rectangle

[parenthesis] top-left position: (x,y) width: w height: h

  • ellipse(x, y, w, h) : Draw a ellipse/oval

[parenthesis] center position: (x,y) width: w height: h

Colors

:

  • (gray) : gray-degree. (from 0-255)
  • (gray, a) : gray-degree, alpha. (all from 0-255)
  • (r, g, b) : red, green, blue. (all from 0-255)
  • (r, g, b, ) : red, green, blue, alpha. (all from 0-255)

*alpha is the degree of transparent.

Setting Styles:

  • noStroke: Without strokes
  • stroke() : Set stroke color
  • strokeWight(w) : Set stroke wight to w
  • noFill: Without fill
  • fill() : Set fill color
  • background() : Fill the entire canvas background

Variables

containers that hold information

  • var x = 10 : Create a new variable "x", and set its value to 10
  • x = x + 3 : Add variable "x" value by 3

System Variables:

  • mouseX, mouseY: x/y position of mouse.
  • random(, b): a random value between and b
  • random(b): a random value between 0 and b
